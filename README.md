# Система «Приемная комиссия»

Проект моделирует работу системы приёма абитуриентов в учебное заведение.

---

## Функционал

- Регистрация абитуриентов на выбранный факультет
- Ввод баллов по вступительным предметам
- Ввод баллов аттестата
- Подсчет общей суммы баллов
- Формирование ведомости абитуриентов
- Определение зачисленных в соответствии с планом набора

---

## Роли в системе

### Абитуриент
- Вводит личные данные
- Указывает факультет
- Вводит результаты экзаменов и аттестата

### Администратор
- Просматривает ведомость
- Определяет зачисленных абитуриентов

---

## Технологии

- Язык программирования: C++
- Система контроля версий: Git
- Хостинг репозитория: GitHub
